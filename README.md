# ya-schedule-api

## Start 

#### Установить node.js

https://nodejs.org

### Установить все модули
> npm install

#### Собрать проект dev (/dist)
(build)
> npm run webpack 

или

(watching)
> npm run webpack-w

#### Запустить локальный сервер (в другой консоли, если webpack в режиме watching)

> npm start

#### Доступ по адресу:
http://127.0.0.1:9250
